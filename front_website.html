🌐 GrabDeals Web (React + Firebase) - Setup & Deployment Guide

This guide helps you set up, configure, and run the GrabDeals web app built using React + TypeScript + Firebase. It’s designed for non-technical users and works on both localhost and shared hosting servers.

📑 Table of Contents

1. Introduction 2. Prerequisites 3. Install Environment 4. Firebase Setup 5. Configure Project 6. Run Locally 7. Build for Production 8. Upload to Shared Hosting 9. Troubleshooting

1. 📘 Introduction

This guide assumes you have received the full GrabDeals website project in a zip file. The app is built with React (TypeScript) and uses Firebase (Realtime Database & Auth).

2. ✅ Prerequisites

  • Node.js and npm (install from nodejs.org)
  • Code Editor (VS Code recommended)
  • Firebase project access (or use the one provided)
  • FTP access or File Manager for shared hosting

3. 🧰 Install Environment

  1. Download and install Node.js
  2. Extract the zipped project folder
  3. Open terminal/command prompt
  4. Navigate to the project folder: 
    cd path/to/project
  5. Install dependencies: 
    npm install

4. 🔥 Firebase Setup

  1. Create a Firebase project at console.firebase.google.com
  2. Enable Realtime Database & Authentication (Email/Password)
  3. Create a user for testing 
    Email: test@example.com
Password: 123456
  4. Go to Project Settings > Web App > Register App > Get config
  5. Copy config into: 
    src/config/firebase.ts

5. ⚙️ Configure Project

  • Open src/config/firebase.ts
  • Replace firebaseConfig with your own values from Firebase
  • Save and close

6. ▶️ Run Locally

  1. Start local dev server: 
    npm run dev
  2. Open browser at http://localhost:5173
  3. Login using your Firebase test user

7. 📦 Build for Production

  1. Run the build command: 
    npm run build
  2. This creates a dist folder with all static files

8. 🌍 Upload to Shared Hosting

  1. Log into your shared hosting control panel (like cPanel)
  2. Open File Manager or connect via FTP
  3. Navigate to public_html or your site root
  4. Upload contents of dist folder
  5. Ensure index.html is directly in public_html
  6. Your site should now be live!

9. 🧯 Troubleshooting

  • Blank screen: Check Firebase config and hosting path
  • 404 errors: Ensure you're not nesting dist inside another folder
  • Login fails: Verify Firebase rules and test user credentials
  • JS/CSS not loading: Confirm all assets are uploaded and accessible
